#### 创建markdown文档

创建文档

> 文章标题为LOVE
>
> ```
> hexo new "LOVE"
> ```

创建草稿文档

> ```
> hexo new draft "LOVE"
> ```
>
> 最后的网站是不会生成草稿文章的

创建纯页面 (网页页面)

> ```
> hexo new page "LOVE"
> ```

创建文章页面

```
hexo new post "LOVE"poe
```



#### 生成网页

在scaffolds中有模板

命令中的draft、page都来自于scaffolds,

根据模板生成对应格式的页面

```bash
hexo g  # 生成页面到public
```
